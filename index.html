⇅ Sync Cloud

Sincronizza i dati tra PC e iPhone via GitHub Gist. Gratuito, nessun server.

Come configurare →
1. Vai su github.com → Settings → Developer settings
   → Personal access tokens → Tokens (classic)
   → Generate new token, spunta solo gist

2. Vai su gist.github.com, crea un nuovo Gist:
   Nome file: agenda_data.json
   Contenuto: {} · Scegli Create secret gist

3. Copia l'ID dalla URL del Gist
   (es. gist.github.com/user/abc123)

4. Incolla Gist ID e Token qui sopra e clicca Salva & Sync
5. Ripeti la configurazione sull'altro dispositivo con gli stessi valori
